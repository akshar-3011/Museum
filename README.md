# Museum of Mango

An archival collection of things too small to keep anywhere else.
This interactive museum contains 5 total rooms featuring memories, photos, letters, and videos.

## Getting Started

To run locally in development mode:

```bash
npm install
npm run dev
```

To build for production:

```bash
npm run build
npm run start
```

## Status
The project is complete and ready for production deployment. All features, storytelling elements, and museum rooms have been finalized. The app is optimized for production.
