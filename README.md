# Museum of Mango

An archival collection of things too small to keep anywhere else.
This interactive museum contains 5 total rooms featuring memories, photos, letters, and videos.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
